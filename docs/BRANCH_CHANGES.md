# 当前分支修改内容

分支：`codex/restdeck-runner-response-cookie-openapi-vars`

## 主要功能

- 新增 OpenAPI 导入能力，可从 OpenAPI 文档生成集合、请求、参数和请求体。
- 增强导入入口，支持 Postman、cURL、Fetch、OpenAPI 等常见来源。
- Runner 页面重新设计，支持选择集合或单个请求执行，展示等待、运行、通过、失败、跳过等状态，并支持复制和导出报告。
- 响应区增加 Cookie 展示，设置页增加 Cookie Jar，可查看、清空和删除本地 Cookie。
- 新增响应变量能力，可从响应中选择 JSONPath 并保存为变量。
- 请求发送增强 Cookie 处理，请求结果会记录实际请求地址。

## 体验优化

- 使用更轻量的设置页布局，设置项分组展示，一行可放置两个设置项。
- 优化下拉框、菜单、按钮、表格、搜索框、Runner、Cookie Jar 等界面细节。
- 表格中的输入框、选择框、Cookie 表格和 Runner 结果列表修复对齐与间距问题。
- 隐藏实时功能入口，保留后端能力但不在主导航展示。
- 请求名称支持点击直接编辑，失焦后自动退出编辑。
- 请求和环境改为即时保存，不再依赖手动保存按钮。
- Form data 的表格/文本切换按钮改为更贴切的“文本编辑 / 表格编辑”。

## 数据与网络

- 数据目录固定为程序同级 `Data` 文件夹。
- 支持默认代理、请求独立代理和代理排除规则。
- 请求级代理配置去掉代理排除项，仅保留是否继承、禁用或自定义代理。
- 环境变量支持静态值、时间戳和响应 JSONPath 来源。

## 右键菜单与代码生成

- 请求列表增加右键菜单，支持生成代码、置顶、复制、删除、导出。
- 生成代码弹窗支持 cURL、PowerShell cURL、JavaScript fetch、Node.js fetch、Python requests、Go、C#、Java 等格式。
- 环境列表支持右键重命名和删除，全局变量作为左侧特殊项展示。

## 构建与资源

- 版本号更新为 `0.3.0`。
- 更新应用图标，并启用 Windows 窗口/任务栏图标显示。
- README 已重写为功能介绍，并补充浅色和暗色界面截图。
