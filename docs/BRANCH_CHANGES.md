# 当前分支修改内容

当前分支：`codex/restdeck-debug-preview-runner-imports`

统计范围：相对 `codex/restdeck-runner-response-cookie-openapi-vars` 的新增修改。

版本：`0.4.0`

## 导入导出

- 新增 Swagger URL 导入：输入 Swagger/OpenAPI 地址后直接拉取并导入集合。
- OpenAPI 导入支持选择文档中的 server 地址。
- 新增 HAR 导入导出能力，便于和浏览器网络记录互通。
- 导入弹框布局改轻，Swagger URL 弹框缩小高度，示例内容改为 placeholder，用户输入后自动隐藏。
- 导出结果支持保存到文件。

## 响应查看

- 合并响应搜索框和 JSONPath 查询框。
- 输入 `$` 时显示 JSONPath 下拉候选，支持上下键切换高亮项，回车直接查询。
- 非法或普通 `$` 开头内容按搜索处理，例如 `$123`。
- JSONPath 查询结果直接显示在响应内容区域，并支持美化展示。
- “复制结果”按钮始终可用。
- “生成变量”改为轻量弹出框，输入变量名后确认创建。
- 响应上方工具条固定在顶部，滚动条只作用于响应内容区。
- 响应分割线精简为单条，并支持拖动调整请求区和响应区高度。

## 环境变量

- 环境变量编辑页从表格改为左侧变量列表 + 右侧详情编辑。
- 变量列表和详情区域填充底部空间，变量较多时单项高度不再被挤压。
- 变量项焦点黑框改为更轻的聚焦样式。
- 响应 JSONPath 变量的测试结果改为只读输入框。

## 请求与代理

- 请求方法下拉框加宽，确保 `POST` 等方法名完整显示。
- 请求标题只在宽度不足时显示省略号。
- 请求级代理移除代理排除项。
- 自定义代理未填写 URL 时按不使用代理处理，不再提示错误。

## 构建与修复

- 版本号升级到 `0.4.0`。
- 修复 Wails 生成绑定时反复输出 `Not found: time.Time` 的问题，时间字段在前端模型中改为 `string` 类型。
- 重新生成 Windows 可执行文件：`build/bin/RestDeck.exe`。
