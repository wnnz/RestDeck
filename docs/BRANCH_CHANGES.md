# 当前分支修改内容

分支：`codex/restdeck-debug-preview-runner-imports`

版本：`0.4.0`

## 主要功能

- 新增 OpenAPI/Swagger URL 导入能力，可从接口文档生成集合、请求、参数和请求体。
- 增强导入导出入口，支持 Postman、cURL、Fetch、HAR、OpenAPI 等常见来源，并支持集合导出到文件。
- Runner 页面重新设计，支持选择集合或单个请求执行，右侧展示等待、运行、通过、失败、跳过等状态，并支持复制和导出报告。
- 响应区增加 Cookie 展示，设置页增加 Cookie Jar，可查看、清空和删除本地 Cookie。
- 响应查询合并搜索与 JSONPath：输入 `$` 显示下拉候选，上下键切换候选，回车直接在响应内容中展示查询结果。
- 支持从有效 JSONPath 生成环境变量，使用轻量弹出框输入变量名；普通 `$123` 等非 JSONPath 内容按搜索处理。
- 请求结果记录实际请求地址，历史列表使用请求地址和实际请求文本展示。

## 请求与集合

- 请求列表增加右键菜单，支持生成代码、置顶、复制、删除、导出。
- 生成代码弹窗支持 cURL、PowerShell cURL、JavaScript fetch、Node.js fetch、Python requests、Go、C#、Java 等格式。
- 集合侧边栏增加固定的新建请求入口，导入入口收敛到集合菜单。
- 请求名称支持点击直接编辑，按内容自适应宽度，失焦后自动退出编辑。
- 请求改为即时保存，不再依赖手动保存按钮。
- Form data 的“界面模式”改为更贴切的“表格编辑 / 文本编辑”，并支持文本项和文件项。

## 环境与变量

- 环境列表支持新增、右键重命名和删除，全局变量作为左侧特殊项展示。
- 环境改为即时保存，不再依赖手动保存按钮。
- 环境变量支持静态值、时间戳和响应 JSONPath 来源。
- 环境变量页面从表格改为左侧变量列表 + 右侧详情编辑，列表和详情区域填充剩余高度，变量较多时不再挤压单项高度。
- JSONPath 测试结果改为只读输入框，便于复制和查看。

## 数据与网络

- 数据目录固定为程序同级 `Data` 文件夹，不再依赖旧用户目录。
- 支持默认代理、请求独立代理和代理排除规则。
- 请求级代理配置去掉代理排除项，仅保留继承、禁用或自定义代理。
- 自定义代理未填写地址时按不使用代理处理，不再报错。
- 代理、请求发送、变量解析和导入导出相关后端逻辑已补充测试。

## 界面体验

- 使用更轻量的设置页布局，设置项分组展示，一行可放置两个设置项。
- 主题色恢复为橙色，明暗主题保持一致的品牌强调色。
- 优化下拉框、菜单、按钮、表格、搜索框、Runner、Cookie Jar、响应工具栏等界面细节。
- 所有菜单项统一左对齐，右键菜单固定弹出到光标右下角。
- 表格中的输入框、选择框、Cookie 表格和 Runner 结果列表修复对齐与间距问题。
- 方法下拉框加宽，确保 `POST` 等方法名完整显示。
- 请求标题仅在空间不足时显示省略号。
- 响应上方分割线可拖动调整高度，并精简为单条分割线。
- 隐藏实时功能入口，保留后端能力但不在主导航展示。
- 更新应用图标，并在程序左上角、Windows 窗口和任务栏中显示。
